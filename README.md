# AI-TUTOR-FOR-UNPRIVILEGED-STUDENTS

Voice Assistant AI Tutor

Voice Assistant AI Tutor is a web-based application designed to help underprivileged students learn in an interactive and accessible way. By leveraging speech recognition and text-to-speech technologies, the app acts as a personal tutor that listens to students' questions and responds with simple, spoken answers.
