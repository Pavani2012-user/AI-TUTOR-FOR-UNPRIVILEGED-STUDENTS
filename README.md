# AI-TUTOR-FOR-UNPRIVILEGED-STUDENTS

Voice Assistant AI Tutor

Voice Assistant AI Tutor is a web-based application designed to help underprivileged students learn in an interactive and accessible way. By leveraging speech recognition and text-to-speech technologies, the app acts as a personal tutor that listens to students' questions and responds with simple, spoken answers.


---

🔧 Features

🎤 Voice Input – Ask questions using your voice

🗣 Voice Output – The tutor speaks back with an answer

🌐 Web Interface – Clean and mobile-friendly user interface

🧠 Basic AI Responses – Supports pre-defined educational Q&A

📦 Frontend & Backend – Built using HTML, CSS, JavaScript, Node.js, and Express



---

🏗 Technologies Used

Frontend: HTML, CSS, JavaScript, Web Speech API

Backend: Node.js, Express.js, Body-Parser, CORS

No database needed (for demo purposes; easy to extend)



---

🚀 How to Run

1. Clone the repository:

git clone https://github.com/your-username/voice-assistant-ai-tutor.git


2. Navigate to the project folder:

cd voice-assistant-ai-tutor


3. Install dependencies:

npm install


4. Start the server:

npm start


5. Open your browser and visit:

http://localhost:3000
